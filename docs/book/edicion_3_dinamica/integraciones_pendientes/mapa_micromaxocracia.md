## Resumen Ejecutivo

La **MicroMaxocracia** es el sistema de equidad doméstica que constituye la **Capa 3** de la arquitectura maxocrática. Hace visible el trabajo doméstico invisible mediante el Vector de Huella Vital (VHV), rituales estructurados y un sistema de niveles de adopción gradual. Prepara a las familias para participar en una civilización maxocrática más amplia.

## Conceptos Clave para Integrar

### 1. El Problema de la Invisibilidad
- Trabajo doméstico no reconocido ni valorado
- Ratio 2,2:1 (mujeres vs hombres en horas semanales)
- Proyección: equidad plena no hasta ~2066 al ritmo actual
- **Capítulo sugerido:** 16 (MicroMaxocracia)

### 2. Principios Rectores
- Lealtad a la Verdad
- Transparencia Radical
- Verificación Mutua
- Negociación Democrática
- **Capítulo sugerido:** 16 (fundamentos)

### 3. Arquitectura de Tres Capas
- **Capa 1:** Dato Objetivo (VHV universal)
- **Capa 2:** Valoración Social (ponderaciones culturales)
- **Capa 3:** Ritual y Simbolismo (ceremonias y prácticas)
- **Capítulo sugerido:** 16 (arquitectura del sistema)

### 4. Vector de Huella Vital Doméstico (VHV)
```
VHV = Tiempo × Factor_Esfuerzo × Factor_Carga_Mental × Factor_Alcance
```
- Ponderaciones estandarizadas por tipo de tarea
- Factor de Intensidad Contextual (FIC) para cuidado
- **Capítulo sugerido:** 5 (métricas), 16 (aplicación doméstica)

### 5. Modelo de Tres Cuentas
```
Equilibrio = α×(CDD/total) + β×(CEH/total) + γ×(TED/total)
```
- **CDD:** Contribuciones Domésticas Directas
- **CEH:** Contribuciones Económicas al Hogar
- **TED:** Tiempo de Energía Disponible
- **Capítulo sugerido:** 16 (modelo económico doméstico)

### 6. Sistema de Niveles de Adopción
- **Nivel 0:** Concienciación (conversaciones)
- **Nivel 1:** Registro informal
- **Nivel 2:** Registro sistemático (ledger)
- **Nivel 3:** Auditoría estructurada
- **Nivel 4:** Optimización digital
- **Capítulo sugerido:** 16 (implementación gradual)

### 7. Rituales Estructurados
- **Check-in semanal** (15 min)
- **Auditoría mensual** (60 min)
- **Revisión anual** (90-120 min)
- **Capítulo sugerido:** 16 (prácticas y rituales)

### 8. Salvaguardas Éticas
- **Escala de Seguridad (ESI):** Verde/Amarillo/Rojo
- **Protocolo de Desintoxicación:** ICE, IDB, IDP
- **Trabajo Emocional No-Indexable**
- **Capítulo sugerido:** 16 (limitaciones y protecciones)

### 9. El Hogar como Laboratorio
- Célula de transformación civilizatoria
- Escuela primera de ciudadanía maxocrática
- Preparación para participación en sistema macro
- **Capítulo sugerido:** 16 (visión y propósito)
