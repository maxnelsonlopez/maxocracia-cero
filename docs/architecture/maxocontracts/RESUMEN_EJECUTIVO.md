## El Problema: Contratos Actuales Generan Pobreza

### Contratos Legales Tradicionales
- ❌ Jerga ofuscada → poder asimétrico
- ❌ Rígidos → litigios costosos
- ❌ Post-conflicto → daño ya ocurrió
- ❌ "Todo o nada" → sin negociación modular

### Smart Contracts Convencionales
- ❌ Técnicos → inaccesibles para público
- ❌ Inmutables → sin corrección posible
- ❌ Amorales → ignoran axiomas éticos
- ❌ Neutros → sin prevención de daño

### MaxoContracts ✅
- ✅ Lenguaje civil + código verificable
- ✅ Retractación ética (inmutabilidad selectiva)
- ✅ Término-a-término modular
- ✅ Validación axiomática embebida
- ✅ Prevención de daño (monitoreo γ)

> "No puedes crear un contrato explotador aunque lo intentes."

## Los 5 Bloques Fundamentales

Los usuarios **no escriben código**. Arrastran bloques pre-validados:

### 1. **ConditionBlock** (Si-Entonces)
```
Si [evento_verificable] entonces [acción]
```

### 2. **ActionBlock** (Ejecutar)
```
Ejecutar [transferencia/acceso/servicio]
```

### 3. **GammaProtectorBlock** (Monitoreo γ)
```
Si [sufrimiento > umbral] entonces [retractar automáticamente]
```

### 4. **SDVValidatorBlock** (Dignidad Mínima)
```
Validar que [ninguna parte cae bajo SDV]
```

### 5. **ReciprocityBlock** (Intercambio Justo)
```
Verificar que [intercambio es balanceado en VHV]
```

**Construcción**:
```
Usuario → Drag-and-drop bloques → Sistema genera código → 
Validación axiomática → Deployment blockchain
```

## Aceptación Término-a-Término

A diferencia de contratos "todo o nada", MaxoContracts permiten negociación modular:

### Ejemplo: Contrato Laboral

**Usuario recibe**:
```
[T1] Salario 48h/semana                    ✓ ACEPTO
[T2] Jornada lunes-viernes                 ✗ RECHAZO
[T3] Beneficios SDV (salud, educación)    ✓ ACEPTO
[T4] Arbitraje interno compañía            ✗ RECHAZO
```

**Usuario propone**:
```
[T2] Jornada lunes-jueves (40h)
[T4] Mediación cohorte comunitaria
```

**Sistema evalúa**:
- Simula escenarios con γ (índice de bienestar)
- Calcula impacto VHV
- Propone compromiso automático

**Resultado**:
```
"Se sugiere:
 - Salario 48h/semana ✓
 - Jornada lunes-jueves (40h) ✓
 - Beneficios SDV completos ✓
 - Mediación cohorte ✓
 
 γ_empleador = 1.3 (satisfecho)
 γ_empleado = 1.5 (mejor aún)"

DECISIÓN: [ACEPTO] [CONTRAOFERTA] [CANCELAR]
```

**Impacto**: Negociación colaborativa, no adversarial.

## Validación Axiomática Embebida

Cada MaxoContract verifica automáticamente:

### Axiomas Temporales (T0-T13)
- ✅ T0: Tiempo finito respetado
- ✅ T7: γ ≥ 1 (minimizar daño)
- ✅ T9: Reciprocidad justa
- ✅ T13: Adaptabilidad ante hechos nuevos

### Axiomas de Verdad (1-8)
- ✅ Axioma 4: Transparencia radical
- ✅ Axioma 6: Sin ofuscación
- ✅ Axioma 7: Verificable

### Protecciones Vitales
- ✅ SDV nunca violado
- ✅ Reciprocidad verificable
- ✅ Lenguaje civil (Flesch-Kincaid <8)

**Garantía**: Si algún chequeo falla → **CONTRATO RECHAZADO** con explicación clara.

## Retractación Ética con Compensación

Los contratos pueden retractarse si emergen hechos vitales nuevos:

### Proceso de 4 Fases

**Ejemplo**: Contrato laboral 60h/semana, γ detectado = 0,6

#### 1. Solicitud (Usuario)
- Presenta evidencia (logs TVI, reportes salud)
- Sistema registra solicitud

#### 2. Pre-Validación (Oráculo Sintético, <5 segundos)
- Detecta: "Violación Axioma T7"
- Extrae: Logs TVI muestran 65h/semana real
- Verifica: Salario no cubre SDV después de 65h
- Recomendación: "Pausar + renegociar"

#### 3. Validación Humana (Oráculo Humano, 24-48h)
- Votación comunitaria
- Revisión de evidencia
- Decisión: **APROBADO**

#### 4. Ejecución Automática
- Contrato **PAUSADO** (sin penalidad para empleado)
- Compensación: **2 Maxos** (daño vital sufrido)
- Renegociación: 48h/semana + SDV mejorado

**Resultado**:
- ✅ Reparación en 48h (vs juicio en 2+ años)
- ✅ Ambas partes aprenden
- ✅ Sistema mejora continuamente
- ✅ Cero intermediarios costosos

### Anti-Gaming (Prevención de Abuso)

**Costo Progresivo**:
- 1ª retractación: Gratis
- 2ª retractación: 0,5 Maxos
- 3ª retractación: 2 Maxos
- 4ª+: Revisión comunitaria obligatoria

**Protección de Buena Fe**:
- Si γ <0,8: Primera retractación sin costo
- Emergencias SDV: Prioridad <24h
- Casos ambiguos: Mediación gratuita

## Decreto Antipobreza: Prácticas Prohibidas

### 1. **Arriendo Infinito**
❌ **Prohibido**: Cobro continuo sin transferencia de propiedad después de recuperar costo.

✅ **Alternativa**: Leasing con Transferencia
```
Vivienda $100,000
Arriendo $1,000/mes
Umbral: 120 meses → Propiedad transferida automáticamente
```

### 2. **Pago Injusto**
❌ **Prohibido**: Remuneración bajo Suelo de Dignidad Vital (SDV).

✅ **Alternativa**: SDV como Piso Salarial
```
SDV Bogotá 2026 = $880 USD/mes
Salario mínimo maxocrático = $5.50 USD/hora
Si mercado paga menos → MaxoContract no se valida
```

### 3. **Externalidades Ocultas**
❌ **Prohibido**: Costos vitales (contaminación, agotamiento) no reflejados en precio.

✅ **Alternativa**: Contabilidad VHV Obligatoria
```
Hamburguesa convencional: $5 (oculta: 2500L agua, metano, deforestación)
Hamburguesa maxocrática: 8 Maxos (~$12, incluye costo ambiental real)
```

### 4. **Transferencias Irreversibles**
❌ **Prohibido**: Transacciones sin retractación ante manipulación/daño.

✅ **Alternativa**: Retractación Ética Validada
- Información falsa probada
- γ <1 resultante
- Caída bajo SDV
- Período de reflexión 24-72h para transferencias >10% TVI mensual
