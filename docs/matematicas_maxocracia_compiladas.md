## II. ECUACIONES TEMPORALES FUNDAMENTALES

### A. Unidad Atómica: Tiempo Vital Indexado (TVI)

**Definición del TVI:**
```
TVI = [ID_Ser, Timestamp_Absoluto, Intervalo_Temporal]
```

Donde:
- `ID_Ser`: Identificador único del ser consciente (humano o digital)
- `Timestamp_Absoluto`: Coordenada temporal universal (ej. 2025-11-06T14:34:15Z)
- `Intervalo_Temporal`: Duración del instante medido (ej. 1 segundo, 1 hora)

**Propiedades:**
- Único: No hay dos TVIs idénticos
- Irrecuperable: Una vez vivido, no puede repetirse
- Sumable: Los TVIs se pueden agregar en TTVI

---

### B. Tiempo Total Vital Indexado (TTVI)

**Fórmula TTVI:**
```
TTVI = Σ(TVI_directos) + Σ(TVI_heredados) + Σ(TVI_futuros)
```

**Componentes:**

1. **TVI Directos:**
   - Tiempo de las personas que trabajaron directamente
   - Diseñador, fabricante, reparador, etc.

2. **TVI Heredados (Amortizados):**
   ```
   TVI_heredado_individual = TTVI_herramienta / N_usos_totales
   ```
   Ejemplo:
   - Martillo con TTVI = 10h
   - Usado en 100 proyectos
   - Cada proyecto hereda: 10h / 100 = 0.1h

3. **TVI Futuros (Proyectados):**
   - Tiempo de mantenimiento
   - Tiempo de reparaciones
   - Tiempo de reciclaje/disposición
   - Se actualiza dinámicamente

---

### C. Cascadas de TVI (Externalidades Temporales)

**Fórmula de Cascada:**
```
TVI_cascada = f(TVIs_bloqueados, n_afectados, criticidad_sistema, factor_infraestructura)
```

**Ejemplo - Accidente de Tráfico:**
```
TVI_cascada = 1h (duración) × 25,000 (personas) × 1.5 (hora pico) × 1.2 (infraestructura crítica)
TVI_cascada = 45,000 TVIs ≈ 5 años de vida humana
```

**Componentes de la función:**
- `TVIs_bloqueados`: Duración del bloqueo temporal
- `n_afectados`: Número de personas impactadas
- `criticidad_sistema`: Factor de importancia (hora pico, ruta crítica)
- `factor_infraestructura`: Multiplicador por tipo de infraestructura

---

### D. Carácter Dinámico del TTVI

**El TTVI se actualiza a lo largo del ciclo de vida:**

Ejemplo - Refrigerador:
```
Año 0 (fabricación):
TTVI = 50h (directos) + 30h (heredados) + 15h (futuros) = 95h

Año 5 (reparación):
TTVI = 50h + 30h + 15h + 8h (reparación real) = 103h

Año 15 (reciclado):
TTVI = 50h + 30h + 8h + 4h + 3h (reciclaje) = 95h
(Vivió más de lo esperado, menor costo final)
```

## III. MÉTRICAS DE COHERENCIA

### A. Coeficiente de Coherencia Personal (CCP)

**Fórmula CCP:**
```
CCP = (ΣΔt_inversión + ΣΔt_gasto_consciente) / (24h - ΣΔt_mantenimiento_base)
```

**Componentes:**
- **Numerador:** Tiempo dedicado a acciones intencionales
  - `Δt_inversión`: Tiempo invertido en el futuro
  - `Δt_gasto_consciente`: Tiempo disfrutado conscientemente en el presente
- **Denominador:** Tiempo discrecional total
  - 24h - tiempo biológicamente necesario (sueño, alimentación)

**Interpretación:**
- `CCP → 100%`: Alta intencionalidad y soberanía
- `CCP → 0%`: Alta fuga de tiempo vital en distracciones

---

### B. Ecuación de Trascendencia

**Fórmula de Trascendencia:**
```
Tr(t) = Σ[(Valor_TVI_Enriquecido - Huella_Vital_Generada) × Impacto]
```

**Componentes:**
- `Valor_TVI_Enriquecido`: Valor positivo generado (conocimiento, arte, sistemas)
- `Huella_Vital_Generada (HV)`: Costo total (TTVI + UVC)
- `Impacto`: Multiplicador de alcance (cuántos seres, cuánto tiempo)

**Significado:**
- Mide el legado neto de una vida en el ecosistema vital colectivo
- `Tr > 0`: Contribución neta positiva
- `Tr < 0`: Huella mayor que contribución

---

### C. Acumulación de Capital Vital

**Fórmula de Acumulación:**
```
K(t) = K(t-1) × (1 - δ) + f(Δt_inv_k)
```

**Componentes:**
- `K(t)`: Nivel de Capital Vital en el momento t
- `δ` (delta): Tasa de entropía o decaimiento natural
- `f(Δt_inv_k)`: Función de inversión temporal

**Tipos de Capital Vital:**
- Conocimiento
- Salud
- Relaciones Sociales
- Habilidades
- Redes de Apoyo

**Principio:**
> El crecimiento requiere inversión consciente para contrarrestar el decaimiento natural.

## IV. VECTOR DE HUELLA VITAL (VHV)

### A. Definición Fundamental

**Fórmula VHV:**
```
VHV = [T, V, R]
```

**Dimensiones:**
- **T**: Componente Temporal (TTVI)
- **V**: Componente Vital (UVC)
- **R**: Componente de Recursos Finitos

> El VHV no es un precio. Es un hecho verificable.

---

### B. Componente V: Unidades de Vida Consumidas (UVC)

**Fórmula UVC Ponderada:**
```
V_total = Σ[UVC_base × F_consciencia × F_sufrimiento × F_abundancia × F_rareza_genética]
```

#### 1. Factor de Consciencia

**Fórmula:**
```
F_consciencia = (criterios_cumplidos / criterios_totales)
```

**Criterios Medibles:**
- ¿Sistema nervioso desarrollado? (Sí/No)
- ¿Cerebro que rehuye al dolor? (Sí/No)
- ¿Propiocepción? (Sí/No)
- ¿Aprendizaje complejo? (Sí/No)
- ¿Autoconsciencia? (Sí/No)

**Escala Provisional:**
```
Plantas:           0.1
Insectos:          0.3-0.5
Peces:             0.5-0.7
Aves:              0.7-0.9
Mamíferos:         0.9-1.0
Primates/Cetáceos: 1.0
```

#### 2. Factor de Sufrimiento

**Fórmula:**
```
F_sufrimiento = (Sufrimiento_Real / Sufrimiento_Mínimo_Posible)^α

Donde α > 1 para penalizar exponencialmente el exceso
```

**Criterios Documentables:**
- Espacio disponible vs. recomendado (m²)
- Densidad de población vs. saludable (individuos/m²)
- Alimentación vs. estándares nutricionales
- Acceso a comportamientos naturales
- Método de sacrificio (tiempo de inconsciencia, dolor)

**Ejemplo - Gallinas:**
```
Granja Ética:
- Espacio: 10 m² vs 8 m² requerido = ratio 1.25
- Alimentación: óptima = ratio 1.0
- Sacrificio: aturdimiento efectivo = ratio 1.0
→ F_sufrimiento ≈ 1.1

Granja Industrial:
- Espacio: 0.05 m² vs 8 m² requerido = ratio 0.006
- Alimentación: mínima = ratio 0.5
- Sacrificio: brutal = ratio 5.0
→ F_sufrimiento ≈ 25
```

#### 3. Factor de Abundancia

**Fórmula:**
```
F_abundancia = (Población_Crítica / Población_Actual)^β

Donde:
- Población_Crítica ≈ 10,000 individuos (mínimo viable)
- β: exponente de sensibilidad
```

**Ejemplo:**
```
Gallinas: 25 mil millones actuales
F_abundancia = (10,000 / 25,000,000,000)^0.5 ≈ 0.0006

Si colapsara a 50,000:
F_abundancia = (10,000 / 50,000)^0.5 ≈ 0.45
(¡750 veces más costosa!)
```

#### 4. Factor de Rareza Genética

**Criterios:**
- Gen completamente nuevo = alto valor
- Especie endémica con adaptaciones únicas = alto valor
- Animal de granja genéticamente idéntico = bajo valor

---

### C. Componente R: Recursos Finitos

**Registro Objetivo:**
- Kilogramos de minerales extraídos
- Metros cúbicos de agua dulce consumida
- Litros de petróleo/gas usado
- Hectáreas de tierra ocupada

**Valoración en Maxos:**
```
Valor_R = Cantidad_física × FRG × CS
```

Donde:
- `FRG`: Factor de Rareza Geológica
- `CS`: Criticidad Sistémica

**Factor de Rareza Geológica (FRG):**
- ¿Cuánto queda en reservas accesibles?
- ¿Dificultad de extracción?
- ¿Velocidad de consumo?

**Criticidad Sistémica (CS):**
- ¿Esencial para el SDV?
- ¿Hay sustitutos disponibles?
- ¿Dependencia sistémica?

**Ejemplo - Litio:**
```
Batería para hospital:
- FRG_litio: Alto
- CS: Muy alta (crítico para energía renovable)
→ Costo relativo bajo (uso crítico justificado)

Juguete electrónico desechable:
- FRG_litio: Alto
- CS: Muy baja (no esencial)
→ Costo prohibitivo (uso frívolo)
```

---

### D. Huella Vital (HV) Integrada

**Fórmula Conceptual:**
```
HV = f(TTVI, ΣUVC_consumidas, ΣUVC_afectadas)
```
